# TailorTalk 🧵🤖

TailorTalk is an AI-powered assistant that integrates with Google Calendar to:
- Book meetings with natural language
- Check free slots
- Give links to Google Meet


---

### ⚙️ Tech Stack
- FastAPI + Uvicorn (Backend)
- LangChain + OpenAI/Groq
- Streamlit (Frontend)
- Google Calendar API
